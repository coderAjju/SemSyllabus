C++ vs Java

Points C++ Java
1. Performance C++ ek compiled language hai, jo ki machine code mein directly translate hoti hai, isse execution time kam hota hai. Java ek interpreted language hai jiska matlab hai ki code bytecode mein hota hai, aur phir JVM (Java Virtual Machine) usko interpret karta hai
2. Memory Management C++ mein aapko memory manually allocate aur deallocate karni padti hai (using 'new' aur 'delete'). Java automatic garbage collection provide karta hai, iska matlab hai ki developer ko memory management ke liye explicitly code likhne ki zarurat nahi hoti.
3. Syntax C++ ki syntax complex ho sakti hai, especially beginners ke liye. Java ka syntax C++ se thoda sa simple hai
4. Platform Independence C++ code ko specific platform ke liye compile karna padta hai, isliye ek platform par likha gaya C++ code doosre platform par nahi chal sakta. Java platform-independent hai kyunki bytecode ko kisi bhi platform par chalaya ja sakta hai, jabki C++ platform-dependent hoti hai.
5. Use Cases C++ system-level programming, game development, aur performance-critical applications ke liye zyada use hoti hai. Java enterprise-level applications, web development, mobile applications, aur cross-platform development ke liye zyada use hoti hai.

Java and internet and WWW

Java Internet aur World Wide Web (WWW) mein kaafi important bhoomika nibhati hai. Yahan kuch aise kshetra hain jahan Java ko aam taur par Internet aur WWW ke sambandh mein istemal kiya jata hai:

1. Server-Side Development: Java ko server-side programming mein istemal karke majboot aur scalable web applications banaye ja sakte hain. Java-based frameworks jaise ki Spring aur Java EE (Enterprise Edition) server-side logic banane mein madad karte hain.e

2. Servlets aur JSP (JavaServer Pages): Servlets aur JSP dynamic web applications banane ke liye istemal hone wale Java technologies hain. Servlets server-side logic handle karte hain, jabki JSP HTML pages ke andar Java code ko embed karne ki anumati deti hai dynamic content generate karne ke liye.

3. JavaServer Faces (JSF): JSF ek Java web application framework hai, jo component-based user interfaces banane mein madad karta hai.

4. Applets: Pehle, Java applets ko web pages par interactive aur dynamic content banane ke liye istemal kiya jata tha. Lekin, JavaScript jaise alternative technologies ke aane se applets ab bahut kam istemal hote hain.

5. Java Database Connectivity (JDBC): JDBC ek Java-based API hai jo Java applications ko relational databases ke saath interact karne mein madad karta hai. Yeh web applications ke liye data store aur retrieve karne mein mahatvapurna hai.

Java support systems

Java "support systems" ka matlab hai ki Java ek aisi programming language hai jo bahut se vibhinn prakar ke systems aur environments ko support karti hai. Yeh support systems kai tariko se hota hai, jinme kuch mukhya hain:

1. Platform Independence: Java ek platform-independent language hai, iska matlab hai ki aap Java code ko ek bar likh kar kisi bhi platform par chala sakte hain bina use dobara modify kiye. Yeh feature Java ko versatile banata hai, kyunki aap apne code ko alag-alag operating systems aur devices par easily deploy kar sakte hain.

2. Portability: Java ka code portable hota hai, jisse aap use ek jagah se doosri jagah aasani se shift kar sakte hain. Iske liye aapko code ko dubara likhne ki zarurat nahi hoti, sirf Java Virtual Machine (JVM) ko target platform par install karna hota hai.

3. Versatility: Java bahut se alag-alag types ke applications aur systems ke liye use hoti hai, jaise ki web applications, mobile applications (Android development), enterprise-level applications, scientific applications, aur bahut kuch. Iski bahut badi library ecosystem hoti hai jo developers ko diverse tasks mein madad karti hai.

4. Web Development: Java ko web development mein bhi kaafi istemal kiya jata hai. Server-side technologies jaise ki Servlets, JSP, aur frameworks jaise ki Spring, Java EE, etc., se web applications aur services banaye ja sakte hain.

5. Mobile Applications: Android operating system Java-based hai, isliye Java ko Android app development ke liye istemal kiya jata hai. Developers Android Studio mein Java ka istemal karke mobile applications bana sakte hain.

6. Networking: Java mein networking aur socket programming ke liye libraries aur APIs mojood hain, jisse aap internet par communication karne wale applications develop kar sakte hain.

Java environment

Java environment ka matlab hai woh sab kuch jahan Java applications run hoti hain.Aise hi kuch IDEs hain, jaise ki Eclipse, NetBeans, aur IntelliJ IDEA developer-friendly environment hai, jo code likhne, compile karne, aur run karne mein madad karta hai.Java environment ko samajhne ke liye, yeh cheezein important hain:

1. Java Development Kit (JDK): JDK ek toolkit hoti hai jo developers ko Java applications likhne aur run karne ke liye provide hoti hai. Isme Java compiler, debugger, aur other tools shaamil hote hain.

2. Java Runtime Environment (JRE): JRE ek aisa environment hai jisme Java applications run hoti hain. Agar aap sirf Java applications run karna chahte hain aur code likhne ki zarurat nahi hai, toh aapko sirf JRE ki zarurat hoti hai.

3. Java Virtual Machine (JVM): JVM ek virtual machine hai jo Java bytecode ko machine code mein translate karke execute karta hai. Yeh platform independence ko ensure karta hai, kyunki aapka code ek bar compile hone ke baad kisi bhi system par JVM ke saath chala ja sakta hai.

Program structure

Java program ka basic structure

// Yahan se program shuru hota hai
public class MyProgram {
// Yahan se main method shuru hota hai
   public static void main(String[] args) {
       // Yahan pe aapka code hota hai
       System.out.println("Hello, World!");
   }
   // Main method yahan khatam hota hai
}
// Program yahan khatam hota hai

  • 'public class MyProgram': Class ka declaration hota hai. Class ka naam MyProgram hai. Har Java program ek class se shuru hota hai.

  • 'public static void main(String[] args)': Main method hota hai, yeh program ki execution ka entry point hota hai. Har Java program ka main method hona zaroori hai.

  • '{}': Braces code blocks ko define karte hain. Yeh batata hai ki ek block of code kahan shuru hota hai aur kahan khatam hota hai.

Tokens

Java ke programs mein tokens basic building blocks hote hain. Yeh hote hain:

  1. 'Keywords': Reserved words jaise ki 'class', 'public', 'static', etc.

  2. 'Identifiers': Aapke code mein aapki khud ki banayi gayi names, jaise ki class ka naam, variable ka naam.

  3. 'Literals': Constants, jaise ki "Hello, World!", 123, 3.14.

  4. 'Operators': Mathematical ya logical operations ke liye, jaise ki +, -, *, /.

  5. 'Separators': Symbols jo separate karte hain different parts ko, jaise ki ;, ,, ().

  6. 'Comments': Explanation ke liye, jaise // ya /* ... */.

Java statement

Java mein statements kuch bhi ek specific action ko perform karte hain. Kuch examples:

// Variable declaration and initialization
int x = 5;
System.out.println("Value of x: " + x);
// Conditional statement
if (x > 0) {
   System.out.println("Positive");
} else {
   System.out.println("Non-positive");
}

  • 'int x = 5;': Variable declaration aur initialization ka statement.

  • 'System.out.println("Value of x: " + x);: Output statement, jo console pe kuch print karta hai.

  • 'if (x > 0) { ... } else { ... }': Conditional statement, jo ek condition ko evaluate karta hai.

Java virtual machine

Java Virtual Machine (JVM) ek software hai jo Java bytecode ko hardware par execute karta hai. Ye ek abstraction layer provide karta hai jisse Java code platform-independent ho jata hai. JVM bytecode ko specific hardware instructions mein convert karta hai, jisse Java programs alag-alag devices par chal sake.

Constants & Variables

Constants

: Constants woh values hote hain jo program mein badal nahi sakti, yani ki unki value ek baar set ho gayi toh woh change nahi ho sakti. Inhe hum final keyword ke sath declare karte hain. Jaise ki:

final int MAX_VALUE = 100;

Ismein MAX_VALUE ek constant hai jiska value 100 hai, aur ise hum program ke kisi bhi part mein modify nahi kar sakte.

Variables

: Variables woh values hote hain jo program mein badal sakti hain. Inhe hum declare karte hain aur unki value ko program ke execution ke dauran change kar sakte hain. Jaise ki:

int count = 5;

Yahan count ek variable hai jiska initial value 5 hai. Hum ise later modify karke kisi bhi integer value se replace kar sakte hain.

Type Casting

1. Implicit Casting(Widening): Implicit Casting mein programmer kisi variable ke type ko change nhi karta hai.Wo automatically hota hai program mein.

int numInt = 100;
double numDouble = numInt; // Implicit casting use

2. Explicity Casting(Narrowing): Explicit Casting ya Narrowing mein programmer apne hisaab se variable ke type ko change karta hai.

double numDouble = 55.67;
int numInt = (int) numDouble; // Explicit casting

Operator

Operator do ya do se jyada operands ke beech mein mathematical operation perform karne ke liye use kiye jata hain.Eske kayi type hote hain hum ek ek karke sabke baarein mein padhenge

1. Arithmetic Operators: They are used to perform simple arithmetic operations on primitive data types. 

            * : Multiplication
            / : Division
            % : Modulo

// Arithmetic Operators
class ArithOps {

2. Unary Operators: Unary Operatos kewal single operand ke saath calculation perform karte hain. 

             Suppose I have a variable 'i'
             1. Post increment operator (i++) :- Eska matlab value pahle initialize hogi phir increment hogi.
                         i++ ko hum i = i + 1 ke form mein samjh sakte hain;
             2. Pre increment operator (++i) :- Eska matlab value pahle increment hogi phir initialize hogi
                         ++i ko hum i = i + 1 ke form mein samjh sakte hain;
             3. Post decrement operator (i--) :- Eska matlab value pahle initialize hogi phir decrement hogi.
                         i-- ko hum i = i - 1 ke form mein samjh sakte hain;   
             4. Pre decrement operator (++i) :- Eska matlab value pahle increment hogi phir initialize hogi
                         --i ko hum i = i + 1 ke form mein samjh sakte hain;
// Unary Operators
class UnaryOps {
   // main function
  public static void main(String[] args)
  {
      // Integer declared
      int a = 10;
      int b = 10;

      // Using unary operators
      System.out.println("Postincrement : " + (a++));
      System.out.println("Preincrement : " + (++a));

      System.out.println("Postdecrement : " + (b--));
      System.out.println("Predecrement : " + (--b));
  }
}

Expressions & Their Evaluation:

Expression: Ek expression ek combination hoti hai operands (values) aur operators ka jo ek value produce karti hai.

int result = 5 + 3; // Yahaan, 5 + 3 ek expression hai jo 8 dega.

Evaluation: Expression ka evaluation uss expression ko solve karke uski final value nikalna hai.

int x = 10;
int y = x * 2; // Yahaan, expression x * 2 evaluate hokar y ko 20 dega.

Decision making

1. If-Else Statement: If-Else ka use hota hai jab aapko kisi condition ke basis par kuch execute karna ho.

int age = 18;
if (age >= 18) {
    System.out.println("You can vote!"); // Agar age 18 ya usse zyada hai, toh yeh print hoga.
} else {
    System.out.println("You are not eligible to vote."); // Warna yeh print hoga.
}

2. else-if statement: Use the else if statement to specify a new condition if the first condition is false.

int num = 10;
if (num > 10) {
    System.out.println("Number is greater than 10");
} else if (num < 10) {
    System.out.println("Number is less than 10");
} else {
    System.out.println("Number is equal to 10");
}

Branching

Switch Statements: Switch ka use hota hai jab aapko ek variable ke alag-alag values ke basis par alag-alag cases ko handle karna ho.

int day = 2;
switch (day) {
    case 1:
        System.out.println("Sunday");
        break;
    case 2:
        System.out.println("Monday");
        break;
    // ... other cases
    default:
        System.out.println("Invalid day");
}

Loops

1. For loop: For loop ka use tab hota hai jab aapko ek fixed number of iterations mein kuch execute karna ho.

for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration: " + i);
}

2. While loop: While loop ka use tab hota hai jab aapko ek condition ke basis par kuch execute karna ho.

int count = 0;
while (count < 5) {
    System.out.println("Count: " + count);
    count++;
}

3. Do-while loop: Do-While loop bhi ek condition based loop hai, lekin yeh loop ke body ko kam se kam ek baar toh execute karega.

int num = 1;
do {
    System.out.println("Number: " + num);
    num++;
} while (num <= 5);

Jumps in loops

1. Break statement: Break statement ka use loop ke execution ko turant terminate karne ke liye hota hai.

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // Loop ko yahaan se bahar karega agar i 5 ho gaya.
    }
    System.out.println("Iteration: " + i);
}

2. Continue Statement: Continue statement ka use loop ke current iteration ko chhodkar agle iteration se shuru karne ke liye hota hai.

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // Isse yeh hoga ki i ki value 3 par loop chhodkar agla iteration shuru hoga.
    }
    System.out.println("Iteration: " + i);
}

Labled loop

Labeled Loop: Java mein aap ek loop ko label dekar identify kar sakte ho. Yeh kisi loop ko specify karne mein help karta hai, especially nested loops ke case mein.

outerLoop:
for (int i = 1; i <= 3; i++) {
    innerLoop:
    for (int j = 1; j <= 3; j++) {
        if (i * j > 4) {
            break outerLoop; // outerLoop se bahar nikal jaayega agar i*j > 4.
        }
        System.out.println("i * j: " + (i * j));
    }
}
